# Timer

タイマーアプリです。以下のようなことを思い、自作してみました。

- パソコン操作しながら、ラーメン作りたい！
- パソコンで気軽にタイマーが触りたい！
- シンプルな機能のものが使いたい！

![Startup project](./doc_image/app_screenshot.jpg)


## アプリケーションの動作：

1分または1秒単位で、99分59秒までタイマーをセットすることができます。

タイマー終了後は「ド・ミ・ソ・ミ・ド」と音が鳴ります。

ポップアップなどはありません。


## 使っているもの：

主に以下のライブラリを使用して作成しています。

- Prism
- ReactiveProperty


## 機能：

- タイマー機能のみ
	- 59分59秒までタイマーを設定できます。
